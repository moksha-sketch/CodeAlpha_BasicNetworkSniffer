# Basic Network Sniffer

## Description

This project is a basic network packet sniffer built using Python and Scapy.

It captures live network packets and displays:

- Source IP Address
- Destination IP Address
- Protocol
- Source Port
- Destination Port
- Packet Payload

## Features

- Live packet capturing
- TCP packet detection
- UDP packet detection
- IP address extraction
- Port number analysis
- Payload inspection

## Technologies Used

- Python 3
- Scapy

## Installation

```bash
pip install scapy
```

## Run

```bash
sudo python3 sniffer.py
```

## Output

Example:

```
Source IP      : 192.168.1.5
Destination IP : 142.250.182.14
Protocol       : TCP
Source Port    : 50012
Destination Port : 443
```
